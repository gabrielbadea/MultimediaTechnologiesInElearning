Aceasta pagina exemplifica ordonarea unui sir prin metoda bulelor.

Sortarea prin metoda bulelor este un algoritm simplu de sortare care parcurge în mod repetat lista de intrare element cu element, comparând elementul curent cu cel de după acesta, schimbându-le valorile dacă este necesar. Aceste treceri prin listă sunt repetate până când nu trebuie efectuate schimburi în timpul unei treceri, ceea ce înseamnă că lista a devenit complet sortată.
